A lightweight plugin that displays bio and social media information. Developed for themes by Rescue Themes.

== Description ==

When activated, this plugin will add a widget called 'Rescue About' to your Widgets area. The widget will allow you to enter a title, biographical information, upload an image, and display social media icons.

This plugin is styled for themes by Rescue Themes. You're welcome to use it with other WordPress themes but you may need to fiddle with the styling.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
4. Drag/Drop the Rescue About widget into your sidebar and fill in the desired information.
